# Subset VCF by genomic range

Extract variants within a genomic region from a VCF file.

## Usage

``` r
subset_vcf_by_range(vcf_path, chrom, start, end, output_path)
```

## Arguments

- vcf_path:

  Input VCF path

- chrom:

  Chromosome (e.g., "22" or "chr22")

- start:

  Start position

- end:

  End position

- output_path:

  Output VCF path
