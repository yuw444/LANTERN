# Create ancestry-specific VCF files

Create ancestry-specific VCF files

## Usage

``` r
create_ancestry_vcfs(vcf_path, gt_matrix, pt_matrix, out_african, out_european)
```

## Arguments

- vcf_path:

  Input VCF path

- gt_matrix:

  Genotype matrix

- pt_matrix:

  PT (ancestry) matrix

- out_african:

  Output path for African-specific VCF

- out_european:

  Output path for European-specific VCF
